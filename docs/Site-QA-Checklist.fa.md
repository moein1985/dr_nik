# چک لیست جامع بازبینی سایت و سامانه نوبت دهی

تاریخ: 1405/03/03
نسخه: 1.0
دامنه هدف: https://clinicdrnik.com

## هدف
این فایل برای تحویل به همکار QA تهیه شده تا:
- همه بخش های سایت را مرحله به مرحله بررسی کند.
- همه ایرادهای ظاهری، محتوایی و عملکردی را ثبت کند.
- قابلیت های پیاده شده نوبت دهی را کامل تست کند.
- اقلام ناقص (عکس، ویدیو، لینک واقعی شبکه اجتماعی و ...) را از کارفرما دریافت و پیگیری کند.

## روش استفاده
- برای هر مورد، تیک بزنید.
- اگر موردی مشکل داشت، در انتهای فایل و در بخش گزارش باگ ثبت شود.
- برای هر مشکل، اسکرین شات یا ویدیو کوتاه ضمیمه شود.
